# Frontend - Sistema de Inventario y Ventas

Frontend construido con **Next.js 14**, **TypeScript**, **Bootstrap 5** y **Tailwind CSS**.

---

## Instalación y Configuracion

### 1. Prerrequisitos

- Node.js 18+ instalado
- Backend corriendo en `http://localhost:8101/api`

### 2. Instalar Dependencias

```bash
cd project-tia-sa-front
npm install
```

### 3. Ejecutar

```bash
npm run dev
```

**Frontend:** http://localhost:3000
